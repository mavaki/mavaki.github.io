# My Personal Website

### Development

I was both excited and curious at the sound of my own website. It was the summer of 2023, and I didn't know much about HTML or web development at the time. But my sister told me about GitHub Pages, so that's where I began. I forked the [github/personal-website](https://github.com/github/personal-website) repository, and almost instantly I had a website with my name, profile, and repositories. I was happy with its clean and basic layout, so I began to customize it piece by piece.

I first added hyperlinked icons to my Instagram and Twitter using the built-in social media code the repository's `_config.yml` file. I quickly realized that I wanted to give visitors access to my BoardGameGeek profile as well, so I spent some time familiarizing myself with the associated files. After a couple failed attempts, I was able to get the logo and link working with the help of [Simple Icons](https://simpleicons.org). I decided to add some color to each of the icons, so I naturally turned to the open-source image viewer [gThumb](https://en.wikipedia.org/wiki/GThumb), which gave me the hex color codes I needed.
