# My Personal Website

### Development

I was both excited and curious at the sound of my own website. It was the summer of 2023, and I didn't know much about HTML or web development at the time. But my sister told me about [GitHub Pages](https://pages.github.com), so that's where I began. I forked the [github/personal-website](https://github.com/github/personal-website) repository, and almost instantly I had a website with my name, profile, and repositories. I was happy with its clean and basic layout, so I began to customize it piece by piece.

I first added hyperlinked icons to my [Instagram](https://www.instagram.com/carkvankirk) and [LinkedIn](https://www.linkedin.com/in/mark-van-kirk-95408121b) profiles using the built-in social media code in the repository's `_config.yml` file. I quickly realized that I wanted to give visitors access to my [BoardGameGeek](https://boardgamegeek.com/user/sparkymarky) profile as well, so I spent some time familiarizing myself with the associated files. After a couple failed attempts, I was able to get the logo and link working with the use of [Simple Icons](https://simpleicons.org). I decided to add some color to each of the icons, so I naturally turned to the open-source image viewer [gThumb](https://en.wikipedia.org/wiki/GThumb), which let me extract the hex color codes I needed. *Voilà!*

My excitement was not short-lived! I soon undertook the creation of different webpages for my recent hobbies: [Arch Linux](http://www.mavaki.com/archlinux), [Board Games](http://www.mavaki.com/boardgames), and my [MGB Project](http://www.mavaki.com/mgbproject). I simply crafted my `hobbies.html` file (based on the included `interests.html` file), added some titles, icons, and links, and I was there! I then had to spend a good chunk of time actually filling in those pages with my thoughts and experiences while attaching photos via [Google Photos](https://www.google.com/photos/about).

Although my website is currently in a satisfactory state, I will soon be adding my own blog and, perhaps, a dynamic program or interface that gives the site a little more character. Stay tuned!
